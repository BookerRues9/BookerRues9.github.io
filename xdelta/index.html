XDeltaTool Privacy Policy

Effective date: 16/05/2025

This Privacy Policy applies to the Android application XDeltaTool, developed and published by Booker Dev.

No Data Collection

XDeltaTool is a paid application that does not use ads or collect any personal data. The app works completely offline and does not require any form of login or network access to function.

Functionality

The app allows users to apply and create XDelta patches using their own files. No data is sent to any external servers, and all operations occur locally on the user’s device.

Permissions

XDeltaTool may request access to the file system through the Android Storage Access Framework (SAF) in order to read and write patch-related files. This access is used solely for the app’s core functionality.
