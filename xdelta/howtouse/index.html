How to Use XDeltaTool

Patching a File

To patch a file, go to the Patch tab in the app. Tap the Select Files button. An alert dialog will appear with two options:

  • Select the .xdelta patch file
  • Select the file to be patched

Once both files are selected, tap Patch. A dialog will confirm that patching was successful.

To save the patched file to external storage, press Save Patched File. This opens a SAF window to choose a location.

Creating a Patch

Go to the Create Patch tab. Tap the Select Files button. A dialog appears with two options:

  • Select the modified file
  • Select the original file

Then tap Create XDelta Patch. A dialog will confirm success.

To save the newly created .xdelta patch to external storage, press Save XDelta File to External Storage and use the SAF file picker.

Optional: Clear Cache

After saving, the app may show a dialog offering to clear its cache. This is optional but helps keep the app clean.
