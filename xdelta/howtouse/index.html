Go to the first tab named Patch. Tap on Select Files.

An alert dialog will appear with two options:

  • Select the .xdelta patch file.
  • Select the original file to patch.

After selecting both files, press the Patch button.

An alert will confirm successful patching. Then, press Save patched file to export it to external storage using SAF (file picker).

ðŸ“¦ Creating XDelta Patches

Go to the Create Patch tab. Tap Select Files.

An alert will open with two choices:

  • First, select the modified file.
  • Then, select the original file.

After that, press Create XDelta Patch.

An alert will confirm success. Then press Save xdelta file to external storage to export it with SAF.

Optionally, you may clear the app cache afterward.
